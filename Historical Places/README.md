# Historical Places — Реестр объектов исторической важности

Ведение данного списка осуществляется в соответствии с **ЗК4.7stars**.

Статус Historical Place присваивается только отдельным постановлением, подписанным Президентом. Данный документ является **регистром** действующих объектов.

---

## Порядок включения нового объекта

1. Принимается постановление о присвоении статуса Historical Place (на основании ЗК4.7stars).
2. После подписания Президентом объект вносится в данный список.
3. С момента включения объект приобретает защиту, установленную Статьёй 19 Конституции.

---
